## Purpose

Shell entry point utility.

## How the code works

- Uses `@echo off` to suppress terminal command mirroring.
- Uses `echo` to output a splash screen message to the console (for example: `Welcome to AYOSOFT - Your PCSX2 Regeneration Program!`).
- Uses `pause` to halt execution flow until a hardware interrupt (keypress) is registered.
